# 一个还算能看的简历模板

- 基于nextjs
- 相对算是比较美观的
- 可以切换为light 和dark 风格，也支持跟随系统的
- 打印友好
- 可以直接vercel一键部署 
- 请事先编辑data/profile.ts ，修改成您的个人信息成
- 目前的个人信息是我胡乱编的
